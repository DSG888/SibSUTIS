Сортировка массива

Задание

Создайте форму, состоящую из:

  • текстовых полей для вывода элементов массива;
  • кнопки "Обновить", по нажатию которой массив заполняется новыми случайными значениями;
  • текстовых полей для вывода элементов отсортированного массива;
  • кнопки "Сортировать", по нажатию которой в соотвествующие текстовые поля выводится отсортированный массив.
